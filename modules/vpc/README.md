## VPC Module

Creates a VPC with public and private subnets across multiple availability zones, an Internet Gateway, route tables, and an optional NAT Gateway for private subnet egress.

### Usage

```hcl
module "vpc" {
  source = "../../modules/vpc"

  name_prefix          = "myapp-dev"
  vpc_cidr_block       = "10.0.0.0/16"
  public_subnet_cidrs  = ["10.0.1.0/24", "10.0.2.0/24"]
  private_subnet_cidrs = ["10.0.10.0/24", "10.0.11.0/24"]
  availability_zones   = ["us-east-1a", "us-east-1b"]
  enable_nat_gateway   = false
}
```
